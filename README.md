This is an Allegro program mimicking the Concentration memory game. It plays in a 5 x 5 grid that appears empty except for one box at the bottom right.
This box resets the board when clicked, reshuffling shape locations and removing previously matched pairs.
The shapes are randomized and appear when you click inside the box. It ignores clicking previously matched shapes or outside the grid.
When two shapes appear, if they are matching they remain on the screen. 
If not, the shapes remain on screen for 5 seconds for the user to memorize before going away again. 
Allegro is needed for installation, download ZIP file and extract. It uses primitives, Font, text, and dialog add ons.
